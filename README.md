Apket - APK تو جیبته!

Apket یه ابزار کوچیک و سریع برای استخراج و اشتراک‌گذاری فایل نصبی (APK) برنامه‌های نصب‌شده روی گوشی اندرویدی شماست.

ویژگی‌ها:
• استخراج APK بدون نیاز به روت
• نمایش اندازه، نسخه، تاریخ نصب و نام پکیج
• اشتراک‌گذاری مستقیم با واتساپ، تلگرام، بلوتوث و...
• رابط کاربری مینیمال و زیبا
• کاملاً رایگان و متن‌باز (Open Source)

مناسب برای:
- وقتی می‌خوای یه برنامه رو بدون گوگل‌پلی برای دوستت بفرستی
- بک‌آپ گرفتن از برنامه‌های ناشناخته یا قدیمی
- اشتراک‌گذاری اپ‌های مودشده یا بتا
